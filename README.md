# Sleep Tracker AI MCP

Log sleep duration, quality, and calculate sleep debt and consistency scores.

## Tools

- `log_sleep` — Log a sleep entry with duration and quality.
- `calculate_sleep_debt` — Calculate cumulative sleep debt against a target.
- `get_sleep_trends` — Return average duration and quality over recent entries.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m sleep-tracker-ai-mcp.server
```
